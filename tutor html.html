Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur halaman web. HTML menggunakan tag untuk mendefinisikan elemen-elemen seperti judul, paragraf, gambar, tautan, dan banyak lagi.

Struktur Dasar HTML


<!DOCTYPE html>
<html>
    <head>
        <title>Judul Halaman</title>
    </head>
    <body>
        <h1>Selamat Datang di Halaman Saya</h1>
        <p>Ini adalah paragraf pertama saya.</p>
    </body>
</html>

Tag Penting dalam HTML

  • <h1> hingga <h6>: Digunakan untuk membuat judul dengan berbagai tingkat kepentingan.
  • <p>: Digunakan untuk membuat paragraf.
  • <a>: Digunakan untuk membuat tautan.
  • <img>: Digunakan untuk menampilkan gambar.
  • <div>: Digunakan untuk membuat kontainer untuk elemen lain.
